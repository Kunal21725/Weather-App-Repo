# 🌦️ Weather App

A responsive Weather Application built using **HTML, CSS, and JavaScript** that provides real-time weather information for any city using a weather API.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays current temperature
- 💧 Shows humidity
- 🌬️ Displays wind speed
- 🌥️ Weather condition icons
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and easy to use

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
